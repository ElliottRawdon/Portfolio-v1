Facial Recognition App

Using React, Node.js and a collection of langauges listed above i created a facial recognition application with a custom database & server containing user sign in features & HTTPS/AJAX security measures to alter sensitive data like passwords between the server/database & user end. The application allows you to upload images using a machine learning API developed to recognise faces, the application will then apply a box around the facial area and add a point to the users score displayed above.
